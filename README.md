# Metal Expansion   金属拓展

Metal Expansion is a NeoForge mod for Minecraft that adds new metals, tools, machines, batteries, and early energy mechanics.金属扩展是我的世界的新锻造模型，增加了新的金属，工具，机器，电池和早期的能源机制。

## Current Features   ##当前特性

- Osmium items and tools   -锇物品和工具
- Lead-acid battery with FE-style energy storage- fe式储能铅酸电池
- Hydroelectric generator   -水力发电机

## Requirements   # #要求

- Minecraft: 26.2   我的世界：26.2
- NeoForge: 26.2.x
